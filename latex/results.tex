\subsection{Synchronization and Initial Conditions Control Identifiability}

Figure~\ref{fig:likelihood} directly visualizes how both phase
asymmetry and initial-condition asymmetry affect the likelihood
surface. We generate synthetic data with true connectivity $\theta =
0.05$ and evaluate the likelihood at the true initial conditions
across a grid of $\theta$ values for four phase differences. The CRLB
ratio between synchronized and asynchronous epidemics exceeds 1
throughout the parameter space, often by orders of magnitude
(SI~Fig.~S1).

\begin{figure}[h]
\centering
\includegraphics[width=\textwidth]{../pix/likelihood_surface.pdf}
\caption{Log-likelihood as a function of $\theta$ for four phase
  offsets. Left: similar initial conditions for both regions. Right:
  different initial conditions. Dashed line: true $\theta = 0.05$.
  With similar ICs (left), synchronized epidemics produce a flat
  likelihood; phase asymmetry progressively sharpens the peak. With
  different ICs (right), even synchronized epidemics show a clear peak
  because asymmetric initial states create distinguishable
  trajectories.}
\label{fig:likelihood}
\end{figure}

With similar initial conditions (left panel), the synchronized curve
is nearly flat---an optimizer would return some $\hat\theta$, but the
likelihood barely changes across the parameter space. This is the
regime where an analyst could easily obtain a misleading estimate:
the fit converges, the reconstruction looks reasonable, yet the
parameter is completely uninformed by the data. Increasing the phase
difference progressively restores a peak near the true value. With
different initial conditions (right panel), even perfectly
synchronized forcing produces a clear likelihood peak, because the
differing initial states cause the two epidemics to follow different
trajectories whose interaction depends on $\theta$.

\subsection{Real Data: Connectivity Cannot Be Inferred}

We estimated connectivity $\theta$ and its CRLB for all $\binom{20}{2}
= 190$ state pairs. Figure~\ref{fig:theta_vs_crlb} shows
$\hat{\theta}$ against $\sqrt{\text{CRLB}}$ for each pair.
Representative reconstructions are shown in SI~Fig.~S4.

\begin{figure}[h]
\centering
\includegraphics[width=0.75\textwidth]{../pix/theta_vs_crlb.pdf}
\caption{Estimated connectivity $\hat{\theta}$ vs.\
  $\sqrt{\text{CRLB}}$ for all pairs of the 20 largest US states.
  Dashed line: significance boundary ($\sqrt{\text{CRLB}} =
  \hat\theta / 1.96$). Points above it cannot reject $\theta = 0$ at
  the 5\% level. Red stars: significant pairs. Blue region:
  non-significant. Orange region: significant. Nearly all pairs fall
  in the non-significant region.}
\label{fig:theta_vs_crlb}
\end{figure}

Nearly all state pairs fall above the significance boundary: the CRLB
standard deviation exceeds the estimate divided by 1.96, so the 95\%
confidence interval includes zero. The optimizer converges and returns
$\hat\theta$ values that span $[0, 0.5]$ (SI~Fig.~S1), but the CRLB
reveals that none of these estimates carry statistical support. US
states share both similar seasonal phases and similar initial
conditions---the regime our theoretical analysis identifies as most
problematic for connectivity inference.
