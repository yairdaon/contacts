\section*{Methods}
Our goal is to find a lower bound for estimation of a connectivity
parameter between regions. To this end we formulate a discrete SIR
model and employ the Cramer-Rao lower bound to the connectivity
parameter between two regions.

\subsection*{The Cram\'er-Rao Lower Bound}

The \emph{Cram\'er-Rao Lower Bound} (CRLB) provides a fundamental limit
on estimation precision. For any unbiased estimator $\hat{\phi}$ of a
scalar parameter $\phi$, the CRLB states:
\begin{equation}\label{eq:crlb_statement}
\text{Var}(\hat{\phi}) \geq \frac{1}{J(\phi)}
\end{equation}
where $J(\phi)$ is the \emph{Fisher Information}. The Fisher
information measures how sharply the (log-)likelihood changes with the
parameter $\phi$. When the Fisher information is large, small changes
in $\phi$ produce large changes in the likelihood, so $\phi$ can be
estimated precisely. When the Fisher information is low, the
likelihood is relatively flat in $\phi$, so many nearby values of
$\phi$ explain the data almost equally well, making estimation
intrinsically noisy. So high Fisher info means the likelihood contains
a “narrow peak” around the true parameter; low Fisher information
implies the likelihood has a ``broad, flat peak''.

For a vector of parameters $\boldsymbol{\phi} = (\phi_1, \ldots,
\phi_p)$, the bound generalizes to:
\begin{equation}\label{eq:crlb_vector}
\text{Var}(\hat{\phi}_k) \geq [\mathbf{J}^{-1}]_{kk}
\end{equation}
where $\mathbf{J}$ is the $p \times p$ \emph{Fisher Information
Matrix} with entries
$$
J_{kl} = \mathbb{E}\left[\frac{\partial \ell}{\partial \phi_k}
  \frac{\partial \ell}{\partial \phi_l}\right].
$$
%
It is best to think of $\mathbf{J}$ as statistical precision, so
$\mathbf{J}^{-1}$ is just the a covariance matrix and (e.g.) its
$(1,1)$ entry is the variance of the estimator of $\phi_1$.

The CRLB has a key implication: if $[\mathbf{J}^{-1}]_{kk}$ is large,
parameter $\phi_k$ cannot be reliably estimated \emph{regardless of the
estimation method employed}. No amount of algorithmic sophistication can
overcome a fundamental lack of information in the data.

\subsection*{Optimization}

To validate the CRLB predictions, we perform parameter inference on
synthetic data generated by simulating the model with known parameters
and adding binomial observation noise (Eq.~\ref{eq:obs_binomial}).

We minimize the negative log-likelihood (Eq.~\ref{eq:loglik}) subject
to $\theta \in [0, 0.5]$, $S_i(0), I_i(0) \in [0,1]$, and $S_i(0) +
I_i(0) \leq 1$. The gradient is minus the score function
(Eq.~\ref{eq:score}), with $\partial\mu_i(t)/\partial\phi$ from the
sensitivity equations (Eq.~\ref{eq:sens_mu}). This gradient is computed
exactly in a single forward pass alongside the state and sensitivity
equations.

We use the \texttt{NLopt} library with the SLSQP algorithm for
gradient-based optimization. To handle potential local minima, we
employ a multi-start strategy with $N_0 = 20$ random initial points,
selecting the solution with lowest objective value. Optimization runs
use relative tolerance $10^{-4}$ and are parallelized across CPU
cores.
